tweeting

how to tweet

you can write tweets that will show up on the fart ai twitter account (@fart_ai)! its quite simple! just say tweet followed by whatever you want to tweet! heres an example:
tweet the weather sure is nice today!

note: the current unix timestamp will be automatically inserted.

the weather sure is nice today! 1617648399; Now

— Fart Ai (@fart_ai) April 5, 2021

rules

sorry but sometimes were no fun so of course we added some rules that you must follow!

  • tweets can only be up to 250 characters
  • you cannot use any slurs in your tweets
  • the @ symbol is automatically blocked because we do not want to mention people because spam mentioning can get us suspended

threads/replies

you can start a thread just as easily as you can start a tweet! threads automatically reply to the last sent tweet. we are working on allowing you to reply/start a thread with any tweet

making a thread

use f.reply to start or continue a thread. after you write your message the bot will show you the tweet at the top of the thread in a success message.
example: f.reply gubyguby


rules

threads have the same rules as tweets

retweeting

this feature is still a work in progress but you can retweet stuff with the bot! we don't allow you to like stuff with the bot.

how to retweet

use f.retweet followed by the id of the tweet you would like to retweet
example: f.retweet 1379100759959175172


how to get the id of a tweet

take the url of the tweet and look at the number at the end of it
example: https://twitter.com/fart_ai/status/1379100759959175172
the id of this tweet is 1379100759959175172

account stats

this command can be executed inside any server
this allows you to view the statistics of the twitter account. the bot must of made at least 1 tweet since it was last restarted for this command to work.
use f.twitstats to view
