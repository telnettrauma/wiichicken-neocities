message events

message events are events that happen sometimes when messages are sent. lets #tacoboutit

  • featured messages
  • wishing upon a fish
  • auto reply

featured messages

sometimes your message may get featured. if it does, you will get a message telling you so, and your message will be added to the fart global news channel in fart gamers. if something goes wrong, the bot will send another message telling you that your message sucks, and you won't get featured. the chance of a message getting featured is 1/250.
note: this feature is only currently supported in the #fart channel in the fart gamers discord server

wishing upon a fish

wishing upon a fish is a rare event that happens where you can wish upon a fish. you will get a message telling you that you can. some bots have special messages when they get featured. this can happen in any server or channel. the chances of this happening are 1/10000.

auto reply

there is a chance that whenever a message is sent the bot will reply to it. when it does it picks from a list of messgaes and then sends it. suggest more phrases to wiichicken. if you are a moderator, you can use f.autochat to switch the frequency of messages.

  • mode 1: 1/5 chance
  • mode 2: 1/10 chance
  • mode 3: 1/25 chance
  • mode 4: 1/50 chance
  • mode 5: 1/100 chance
  • mode 6: disabled
Pro Tip: you can quickly change to a specific mode by putting the number after the command. for example: f.autochat 4
