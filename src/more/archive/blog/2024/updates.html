SAME SH!T, NOW ON GiT

February 4, 2024

Sorry about the title, that rhyme was too good not to include.
Version control using Git on my site is now working. This shouldn't mean anything to you, the viewer, but it means a lot to me. This is what I had to do before in order to update my site:

  • Update each page
  • Remember what pages I updated
  • Navigate to each folder in Neocities, click upload, find the same folder on my hard drive
  • Copy and paste the changes for each page into Neocities

Now, this is all I need to do:

  • Update each page
  • Click commit (basically publish)

So yeah, I am very excited about this, especially after all of the trial and error that I went through to get to this point.
This will also allow me to use packages, which could make my site better. I tried using eleventy, but I had too many pages to remake, and I didn't want to go through the hassle of remaking all of them.
