Settings

This section will cover the settings menu that appears in the spacebar menu. This menu only scratches the surface of what you can do. Just know that at the bottom of this menu, there's a second settings menu with more settings, and at the bottom of that settings menu is another settings menu with a TON more settings. This will just provide a basic explanation for what features do. I recommend playing around and just tapping random stuff and seeing what you find.

Menu Settings

When you open the spacebar menu, there are 3 pages of options. To the far right of the first page, there is a quality assurance button, and a pencil button. The pencil button will allow you to customize the spacebar menu, including adding and removing features from it. Here are some of the settings you can enable:

  • topbar unshrinker - if you ever decide to hide the topbar, this will add a button to the menu to quickly restore it
  • old bonus things - adds a few just for fun things
  • nintype version date - shows the version at the bottom of the menu
  • date display - shows the current date in the menu
  • paw bonus things icon - adds a submenu with some fun features
  • search and replace - adds a button that lets you search and replace in the text youve typed
  • pencil mode switcher - this one only works on an ipad. i dont have an ipad, so i cant tell you
  • main modes switcher - allows you to switch the typing mode (slidemode, peckmode)
  • SMS saver - features that you wont need unless you absolutely NEED to make a message as small as possible (you wont)
  • shortcuts - buttons to add custom shortcuts
  • big shortcuts - more custom shortcut buttons, but this time they're big
  • typing tutor - sometimes will let you tap into a tutorial on how to glyde type a word. good for beginners
  • layout adjuster - lets you quickly switch between one handed modes

Basic Settings

  • setup - reruns the first time setup
  • search - lets you search for specific settings
  • mods - shows a list of all settings you have changed

Resize

This will let you change the size of the keyboard. But there's more you can do than just the height. You can choose the size of various elements, skew the keyboard towards one side of the screen, add a split, and more.

  • topbar size - tap the bird in the corner to toggle the size of the topbar
  • 3 line button - menu with presets, as well as the ability to show/hide the number row and apostrophe button. you can also save and load layouts.

Visuals

Let's you change visual indicators, appearance, and effects.

Indicators

  • big touch circles - makes the circles when you tap a key larger
  • fading slide trails - when glide typing, the trail will fade out
  • letterflow path - once you finish typing a word, a line will draw out the path to glide type the word
  • character preview - shows a little popup saying what key you pressed. the ios keyboard does this by default, but i feel its distracting on this Nintype
  • bigass word preview - while typing a word, the word will be shown in giant text in the center of the keyboard
